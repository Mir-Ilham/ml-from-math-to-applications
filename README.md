# ML From Math To Applications

A comprehensive, end-to-end **Machine Learning & Deep Learning curriculum** covering everything from foundational mathematics and Python programming to advanced neural networks, computer vision and NLP.

---

## 1. Curriculum Overview

### Phase 1: Python & Mathematics Fundamentals

- **Python:** Structured programming, data types, functions, loops, OOP, file I/O, exception handling, and modules.
- **Mathematics:** Linear algebra, calculus, probability, statistics, and essential math for ML.

---

### Phase 2: Python Scientific Libraries & EDA

- **NumPy:** Arrays, tensors, linear algebra, broadcasting, element-wise operations.
- **Pandas:** DataFrames, data wrangling, handling missing data, merging, and SQL/JSON integration.
- **Data Visualization:** Matplotlib and Seaborn plots for univariate, bivariate, and multivariate analysis.
- **Exploratory Data Analysis:** Automated profiling, preliminary data understanding, univariate/bivariate/multivariate analysis.

---

### Phase 3: Feature Engineering & Classical ML

- **Feature Engineering:** Feature scaling, encoding, handling missing values, outlier detection, PCA, and imbalanced datasets.
- **Core ML Algorithms:** Linear, logistic, polynomial, ridge, lasso regression; KNN, SVM, Naive Bayes, Decision Trees, Ensemble methods, Gradient Boosting, XGBoost, and clustering (K-Means, DBSCAN).

---

### Phase 4: Artificial Neural Networks (ANNs)

- **ANN Concepts:** MLPs, perceptrons, backpropagation, gradient descent, activation functions, weight initialization, regularization, batch normalization.
- **Frameworks:** PyTorch and Keras — building ANNs, autograd, datasets, dataloaders, and hyperparameter tuning (Optuna, Keras Tuner).

---

### Phase 5: Convolutional Neural Networks (CNNs) & OpenCV, Recurrent Neural Netoworks (RNNs)

- **OpenCV:** Basics for image preprocessing.
- **CNN:** Theory, backpropagation, data augmentation, pretrained models, filter/feature visualization.
- **RNNs:** RNN, LSTM, GRU, and advanced sequence modeling architectures.

---

### Phase 6: Transformers & NLP

- **Transformers:** Self-attention, encoder-decoder architecture, and transformer components.
- **NLP Pipelines:** Text preprocessing, word embeddings (Word2Vec), text classification, POS tagging, and real-world NLP projects like Duplicate Question Pairs.

---

## 2. Setup and Installation

### Prerequisites

- Python 3.11.9
- Git

### Step-by-step Setup

1. **Clone the repository**

   ```bash
   git clone https://github.com/Mir-Ilham/ml-from-math-to-applications
   cd ml-from-math-to-applications
   ```

2. **Create and activate a virtual environment** (recommended)

   ```bash
     python -m venv .env
     .env\Scripts\activate
   ```

3. **Install all required packages with exact versions**

   ```bash
   pip install -r requirements.txt
   ```

4. **Launch Jupyter Notebook**

   ```bash
   jupyter notebook
   ```

   or

   ```bash
   python -m notebook
   ```

## 3. How to Use This Repository

- Navigate to the phase folder of your interest.
- Open the corresponding .ipynb file.
- Run cells sequentially
