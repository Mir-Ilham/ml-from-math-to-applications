# 📘 Phase 3: Feature Engineering & Machine Learning Algorithms

This phase focuses on building a strong foundation in **feature engineering** and implementing core **machine learning algorithms**. It covers preprocessing techniques, handling real-world data issues, and understanding both fundamental and advanced models.

---

## 🔧 Feature Engineering

Feature engineering transforms raw data into meaningful inputs for machine learning models.

### 📊 Topics Covered

- **Feature Scaling**
  - Standardization (Z-score normalization)
  - Min-Max Scaling

- **Handling Non-Numerical Data**
  - Label Encoding
  - One-Hot Encoding
  - Ordinal Encoding

- **Handling Missing Data**
  - Deletion methods
  - Imputation (mean, median, mode)
  - Advanced imputation techniques

- **Outlier Detection & Handling**
  - Z-score method
  - IQR method
  - Transformation and capping

- **Feature Construction**
  - Creating new features from existing ones
  - Domain-driven feature generation

- **Principal Component Analysis (PCA)**
  - Dimensionality reduction
  - Variance preservation

- **Imbalanced Data Handling**
  - Oversampling (SMOTE)
  - Undersampling
  - Class weighting
