# Phase 2: Python Scientific Libraries & Exploratory Data Analysis

This phase focuses on learning the **core Python data science stack** used in Machine Learning workflows.

You will learn how to:

- Perform numerical computation with **NumPy**
- Manipulate datasets using **Pandas**
- Visualize data using **Matplotlib and Seaborn**
- Perform **Exploratory Data Analysis (EDA)**
- Build datasets using **APIs and Web Scraping**

# 🔎 Exploratory Data Analysis

<details>
<summary>Click to expand EDA topics</summary>

### Data Understanding

The **7 Preliminary Questions**

1. What are the features in the dataset?
2. What type of data does each feature contain?
3. Are there missing values?
4. Are there outliers?
5. What are the feature distributions?
6. Are variables correlated?
7. Are there patterns affecting modeling?

### Types of Analysis

**Univariate Analysis**

- Histograms
- Distribution plots
- Summary statistics

**Bivariate Analysis**

- Scatter plots
- Box plots
- Correlation plots

**Multivariate Analysis**

- Pair plots
- Correlation matrices
- Heatmaps

### Automated Profiling

- Pandas Profiling / `ydata-profiling`

</details>
